Recon cepat, output neon, klik-to-open, siap dipakai bug hunter.

IP-SCAN adalah aplikasi terminal berbasis Python untuk scan domain/IP: DNS lookup (termasuk host terkait), GeoIP ke peta, RDAP (IP WHOIS), recon HTTP/TLS, pencarian subdomain pasif via CT (crt.sh), dan enumerasi parameter pasif (termasuk hidden fields) + crawl ringan. Halaman ini hanya untuk informasi & cara pakai (aplikasinya dijalankan di terminal).

Fitur Lengkap

  • 01
    DNS Recon Resolve domain: A/AAAA, NS/MX/TXT/SOA/CAA, PTR, host terkait (NS/MX/CNAME/WWW), autodiscover/mail, DMARC/MTA-STS/TLS-RPT.
  • 02
    Subdomain (Pasif) Enumerasi subdomain dari sumber publik (CT/crt.sh + CertSpotter), resolve A/AAAA, lalu ikut masuk ke SCAN.
  • 03
    GeoIP + Map Lookup lokasi IP (fallback provider) dan tampilkan marker real-time di peta (Leaflet) saat scan berjalan.
  • 04
    RDAP / IP WHOIS NetRange, CIDR, Org, status, kontak, referensi registry. Dedup netblock + hilangkan spam (contoh sertifikat panjang).
  • 05
    HTTP/TLS Recon Status/redirect, headers, title, security headers, info sertifikat TLS (SAN/validity/cipher), plus deteksi WAF berbasis header/body (pasif).
  • 06
    Params Discovery (Pasif) Ambil parameter dari query string, URL di HTML/JS, form fields, dan hidden inputs. Tampilkan contoh URL ber-parameter.
  • 07
    Crawl Ringan (GET only) Crawl same-host untuk kumpulkan lebih banyak endpoint & parameter (tanpa fuzzing, tanpa brute force, tanpa POST).
  • 08
    Web Discovery Ambil endpoint low-impact dari robots.txt, sitemap.xml, dan /.well-known/security.txt untuk memperkaya mapping.

Tips: mode komplit pakai --full. Parameter & crawl pakai --params + --crawl. Subdomain CT pakai --ct. Untuk lihat semua contoh, jalankan python ip_scan.py -h.

Cara Pakai

Install & Run 
git clone https://github.com/floryid/IP-SCAN.git
cd IP-SCAN
python ip_scan.py --help
Contoh (Power Mode) 
python ip_scan.py idf.il --full --crawl --crawl-pages 30 --crawl-depth 2 --params-limit 300 --params-urls 60 --delay 0.2
python ip_scan.py example.com --ct --sub-sources ct,certspotter --ct-limit 600 --ct-resolve 250
python ip_scan.py example.com --map
python ip_scan.py 8.8.8.8 --rdap

GitHub Pages tidak bisa menjalankan Python terminal app di browser. Jalankan dari terminal (Windows/Linux/macOS). Map mode akan membuka URL lokal seperti http://127.0.0.1:8000. Kalau warna/link tidak muncul di terminal tertentu, pakai --force-color / --force-links. Untuk parameter discovery yang lebih komplit, tambah --params + --crawl.
